# 图标文件说明

本目录用于存放 tabBar 图标文件。

请添加以下图标文件（建议尺寸：81x81 像素，PNG 格式）：

- home.png / home-active.png - 首页图标
- news.png / news-active.png - 骗局百科图标
- check.png / check-active.png - 风险识别图标
- test.png / test-active.png - 反诈测试图标
- report.png / report-active.png - 举报求助图标

如需使用 emoji 图标作为临时替代，可以修改 app.json 中 tabBar 配置。
